# 开始

- [vite](https://cn.vitejs.dev/)
- [typescript](https://www.typescriptlang.org/zh/)
- [react](https://react.dev/)

## shared

共享模块

```bash
pnpm create vite shared
```

## app

H5 应用

```bash
pnpm create vite app
```

## admin

后台管理系统

```bash
pnpm create vite admin
```

## server

服务端

```bash
pnpm i -g @nestjs/cli

nest new server
```
